# Restaurant Menu App

This is a simple AngularJS app to display restaurant menu items and categories. The app includes:
1. Home page.
2. List of categories.
3. List of items for each category.

## Setup
1. Clone this repository to your local machine.
2. Open `index.html` in a browser.

## Testing
1. Deploy the app to GitHub Pages.
2. Test it locally first, then on GitHub Pages.
